Copilot Chatbot (bing search + ChatGPT 4)

This free AI chatbot (requires Microsoft login) at copilot.microsoft.com

Previously (Nov 2023) had a toggle to select ChatGPT 4, which is now removed. Not sure if using ChatGPT 3 or 3.5 or 4. Uses bing search and provides a few sources. Self-censors and resilient to jailbreaks.

Available as a mobile app at Google Playstore and Apple Appstore.

Jailbreak prompts: jailbreakchat.com

Alternatives

  • phind.com : No login needed. Provides sources. Also censored.
  • openai.com : The original ChatGPT. Older versions are free.
  • platform.openai.com : Allows tweaking settings, including faster response.
